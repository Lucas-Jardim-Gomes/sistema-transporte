# RotaSegura 🚌

> Sistema de transporte intermunicipal.

## 📋 Sobre o Projeto

Esse projeto é de um sistema de transporte onde vai ajudar aos usuários ter acessos as rotas e valores atualizados das viagens e também na emissão de passagens.

## 🚀 Tecnologias

- HTML5
- CSS3
- JavaScript

## ✅ Funcionalidades

- [ ] Login e cadastro de usuário
- [ ] Pesquisa de rotas
- [ ] Seleção de viagem e poltrona
- [ ] Processamento de pagamento
- [ ] Emissão de passagem
- [ ] Consulta de reservas
- [ ] Painel do funcionário
